    【Vue github搜索案例】总结

    谦逊 5/9/2022 Vue案例

    # github案例

    技术栈:Vue+axios+bootstrap
方法: 全局事件总线进行组件间通信

    # 要点:

    - 1.导入bootstrap时需要注意:
    - 使用import导入有严格引入要求,会报错
    - 解决方法:
        - 使用link进行引入
- 2.全局事件总线:
    - 在Vue实例创建之前添加
    - Vue.prototype.$bus = this
    - this.$bus.$on('xxx',Info)    绑定全局事件  
    - this.$bus.$emit('xxx',Info)  触发全局事件
    - 总结:
        - 谁需要数据就给谁绑定事件
        - 谁要传数据就触发绑定事件
- 3.新语法:
    - 合并对象 {...obj1,...obj2} 
    - 模板字符串    `xxxx${变量}`
- 4.设置搜索状态:
    - 是否第一次使用 isFirst
    - 是否正在加载   isLoading
    - 错误时提示     ErrorMsg
    - 数据          users
    - 总结:
        - 需要用对象形式传递
        - 注意不能直接赋值对象,getter、setter会被覆盖

    # list.vue

    <template>
  <div class="row" >
      <!-- 展示用户列表 -->
      <div class="card" v-for="ele in info.users" :key="ele.id">
        <a target="_blank" :href="ele.html_url">
          <img :src="ele.avatar_url" style='width: 100px' />
        </a>
        <p class="card-text">{{ele.login}}</p>		<!-- 用户名 -->
      </div>
      <h1 v-show="info.isFirst">欢迎使用!</h1>
      <h1 v-show="info.isLoading">加载中......</h1>
      <h1 v-show="info.ErrorMsg">{{info.ErrorMsg}}</h1>
  </div>
</template>

<script>
export default {
    // eslint-disable-next-line vue/multi-word-component-names
    name:'List',
    data() {
        return {
           info:{
            //  是否是第一次使用
               isFirst:true,
            //  是否正在加载
               isLoading:false,
            //  是否请求错误
               ErrorMsg:'',
            //  数据
               users:[]
           }
        }
    },
    mounted() {
        this.$bus.$on('getUserLogin',(dataObj)=>{
            // 收到userInfo
            console.log(dataObj);
            console.log('收到了search传过来的数据',dataObj.users);
            // 使用ES6合并对象语法 => 不会清除isFirst属性
            this.info = {...this.info,...dataObj}
        })
    },
}
</script>

    # search.vue

    <template>
  <section class="jumbotron">
      <h3 class="jumbotron-heading">Search Github Users</h3>
      <div>
        <input type="text" v-model="keyword" placeholder="enter the name you search"/>&nbsp;
        <button @click="search">Search</button>
      </div>
  </section>
</template>

<script>
// 导入axios
import axios from 'axios'

export default {
    // eslint-disable-next-line vue/multi-word-component-names
    name:'Search',
    data() {
        return {
            keyword:''
        }
    },
    methods: {
        search(){
        // 更新数据
        this.$bus.$emit('getUserLogin',{isFirst:false,isLoading:true,ErrorMsg:'',users:[]})
           axios.get(`https://api.github.com/search/users?q=${this.keyword}`).then(
               response =>{
                    console.log(response.data.items);
                    this.$bus.$emit('getUserLogin',{isLoading:false,ErrorMsg:'',users:response.data.items})
               },
               error =>{
                  this.$bus.$emit('getUserLogin',{isFirst:false,isLoading:false,ErrorMsg:error.message,users:[]})
               }
           ) 
        }
    },
}
</script>
