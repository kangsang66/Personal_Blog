    【Vue插槽案例】总结

    谦逊 5/11/2022 Vue案例

    # Vue插槽案例

    # 插槽

    - 作用:
  - 接收父组件传递给子组件的html内容,适用于父组件==>子组件
- 分类:
  - 默认插槽
    - <slot></slot>
  - 具名插槽
    - <slot name="data"></slot>
  - 作用域插槽
    - <slot :games="games"></slot>
- 注意点:
  - 1.slot有默认值(当App组件没传递内容时)
  - 2.也可以直接将元素写在slot里面,但需注意样式问题
  - 3.组件名需写成双标签形式

    # 案例知识点

    - 1.组件传值
  - :games="games" =>  子组件向父组件传递
  - data="游戏"    =>   直接给该组件传递值
    - 都需要使用props接收
- 2.v-for,:keys的使用
- 3.{结构赋值} 用于结构子组件向父组件传递数据

    # 默认插槽写法

    App.vue

    <template>
  <div class="container">
    <Category title="游戏" a="瓦大喜哇" b="kashds">
        <ul v-for="(g,index) in games" :key="index">
            <li>{{g}}</li>
        </ul>
      <img src="https://wdlakjdl_01.gitee.io/dist/logo.png" alt="">
    </Category>
    <Category title="爱人">
        <ul v-for="(l,index) in loves" :key="index">
              <li>{{l}}</li>
        </ul>
    </Category>    
    <Category title="音乐">
        <ul v-for="(s,index) in songs" :key="index">
              <li>{{s}}</li>
        </ul>
    </Category>
  </div>
</template>

<script>
// 引入各组件
import Category from './components/Category.vue'

export default {
  name: "App",
  components: { Category },
  data() {
      return {
          games:['虐杀原形','荒野大镖客','LOL','泰坦陨落2'],
          loves:['mother','father','sister'],
          songs:['曹操','杀手','可惜没如果']
      }
  },
}
</script>
<style scoped>
  .container{
    display: flex;
    justify-content: space-around;
  }
  img {
    width: 100%;
  }

</style>

    Category.vue

    <template>
  <div class="category">
      <h3>{{title}}分类</h3>
      <h3>{{a}}</h3>
      <h3>{{b}}</h3>
      <!-- 使用slot默认插槽将该组件特性传入 -->
      <slot></slot>
  </div>
</template>

<script>
export default {
    // eslint-disable-next-line vue/multi-word-component-names
    name:'Category',
    props:['title','a','b']
}
</script>

<style scoped>
    .category{
        width: 200px;
        height: 300px;
        background-color:skyblue;
    }
</style>

    # 具名插槽写法

    Category.vue

    <!-- 给对应元素指定插槽name属性 -->
<img slot="center" src="https://wdlakjdl_01.gitee.io/dist/logo.png" alt="">
    <Category title="爱人">
        <ul v-for="(l,index) in loves" :key="index">
              <li>{{l}}</li>
        </ul>
        <!-- 具名插槽第一种写法 -->
        <div slot="footer" class="box">
          <a href="#">单机游戏</a>
          <a href="#">网络游戏</a>
          <a href="#">手机游戏</a>
        </div>
    </Category>    
    <Category title="音乐">
        <ul v-for="(s,index) in songs" :key="index">
              <li>{{s}}</li>
        </ul>
        <!-- 具名插槽第二种写法 -->
        <template v-slot:footer>
          <a href="#">单机游戏</a>
          <a href="#">网络游戏</a>
          <a href="#">手机游戏</a>
        </template>
    </Category>

    App.vue

        <slot name="center"></slot>
    <slot name="footer"></slot>

    # 作用域插槽写法

    Category.vue

          // 单项绑定数据到对应插槽,类似子传父组件 
      <slot :games="games" :loves="loves" :songs="songs"></slot>

    App.vue

     <Category title="爱人">
        <template v-slot="{loves}">
          <ol>
              <li v-for="(l,index) in loves" :key="index">{{l}}</li>
          </ol>
        </template>
    </Category>    
    <Category title="音乐">
        <template v-solt="{songs}">
          <h4 v-for="(s,index) in songs" :key="index">{{s}}</h4>
        </template>
  </Category>
